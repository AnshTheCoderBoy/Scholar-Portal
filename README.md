# 🎓 Scholar Portal

**Scholar Portal** is a Django-based web application designed to manage and display academic data, such as graduation statistics, scholarship records, and batch-wise student information. Built for institutional use, it provides a simple interface for data entry, browsing, and analysis.

---

## ✅ Key Features

- 🔐 Admin-secured backend (Django Admin Panel)
- 🎓 Store and manage graduation data
- 💰 Maintain scholarship award records
- 📅 Track academic years and batches
- 🔎 Filter/search student data by batch, year, or status
- 📥 Add, update, and delete student records
- 📊 Simple dashboards for data overview (if available)
